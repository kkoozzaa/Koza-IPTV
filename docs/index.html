🌍 Language Note: Although this website and documentation are in English, the Koza IPTV application interface is currently only available in Turkish.

Key Features

Hybrid Multi-Source

Fetches and merges live channels from multiple M3U sources simultaneously for maximum uptime.

Intelligent UI/UX

Minimalist sidebar navigation that stays hidden until you need it. Immersive viewing at its best.

Hold OK Gesture

Unique long-press system to access advanced settings and favorites without cluttering the screen.

Native Performance

Built with AVPlay integration for ultra-fast buffering and native 1080p/4K upscaling.

Technical Specs

Platform Samsung Tizen OS (Smart TV)
Engine Tizen Web API & Samsung Product API
Stack Vanilla JavaScript, HTML5, CSS3

Remote Controls

UP / DOWN: Navigate channels

LEFT / RIGHT: Switch Categories

OK (Short): Play Channel

OK (Long): Open Favorites & Options
